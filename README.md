# 🦀 Rust Projects

A collection of Rust projects built while learning and applying Rust to real-world software engineering problems.

This repository contains experiments, practice projects, backend services, system programming examples, concurrency patterns, networking, and production-oriented Rust applications.

---

## 📚 About

The goal of this repository is to:

- Learn Rust from fundamentals to advanced concepts.
- Build production-quality Rust applications.
- Practice ownership, lifetimes, traits, async programming, and concurrency.
- Explore systems programming and backend development.
- Document interesting Rust patterns and best practices.

# Running Projects

Clone the repository

```bash
git clone https://github.com/<username>/rust-projects.git
```

Move into a project

```bash
cd beginner/project_name
```

Run

```bash
cargo run
```

Build

```bash
cargo build
```

Run tests

```bash
cargo test
```

Check formatting

```bash
cargo fmt
```

Lint

```bash
cargo clippy
```
